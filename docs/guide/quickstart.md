# 快速开始

本节将介绍如何在项目中使用 EarthUI

## LButton 用法

:::demo 使用`size`、`color`、`pain`、`round`属性来定义 Button 的样式。

```vue
<template>
  <LButton @click='click'>按钮</LButton>
</template>

<script>
  export default{
    methods:{
      click(){
        alert(1111)
      }
    }
  }
</script>
```
:::
